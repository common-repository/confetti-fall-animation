Confetti fall animation fireworks celebration plugin for your blog or webpage. Add a delightful falling confetti animation to your website.

==============================================================================

=== How to Use ===
	
1- Goto Wordpress plugin directory 
2- Search for Confetti fall animation plugin. Install and activate
3- Activate the confetti fall animation on HomePage by clicking the buttn or
Use the shortcode [confetti-fall-animation delay="1" time="25"] on any (individual) post or page to start a falling confetti animation.
4- Add shortcode by button in page editing tools
5- Enjoy the animation

===============================================================================
